# Problem 1

Source: [`Problem Set 2026-1.pdf`](../docs/Problem%20Set%202026-1.pdf)

This first problem aims to familiarize you with the conversion between different
numeral systems.

## Problem 1.1

a) Convert the binary number `(10010110)_{B=2}` into a decimal number.

b) Convert the decimal number `(77)_{B=10}` into a binary number.

## Problem 1.2

a) Convert the binary number `(11011001)_{B=2}` into a hexadecimal number
with basis `B = 16`.

b) Convert the hexadecimal number `(4d7a)_{B=16}` into a binary number.
