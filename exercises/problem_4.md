# Problem 4

Source: [`Problem Set 2026-1.pdf`](../docs/Problem%20Set%202026-1.pdf)

Problem 4 revolves around Bitcoin transaction data. Use a block explorer to
answer the following questions, for example
<https://www.blockchain.com/explorer>.

a) Find the transaction that was initiated by the address
`1CGN1zd62UakPw7neohJN8zTJUJ9LT2Ay8` in block `325,001`. What is the
transaction hash?

b) Who mined block `325,001` and what is the Bitcoin address?

c) How many inputs and outputs does the transaction in a) consist of?

d) What is the `pubKeyHash` of the output with index `4`?

e) One output has a different script transaction type than the others. What is
it and how is it used?
