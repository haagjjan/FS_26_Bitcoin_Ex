# Problem 2

Source: [`Problem Set 2026-1.pdf`](../docs/Problem%20Set%202026-1.pdf)

The following English text was encrypted with the monoalphabetic substitution
cipher:

```text
LNCPFUVBLUNF, YKNZCZSBUF, BFQ XHBWL ZNFLWBZLX BWP LPWHX
LSBL POPWTNFP UX QUXZIXXUFM. LSP ZPFLPW GNW UFFNOBLUOP
GUFBFZP BL LSP IFUOPWXULT NG YBXPK NGGPWX LSP IFUAIP ZSBFZP
LN ZNFOPWL LSPNWPLUZBK CFNEKPQMP BFQ UFFNOBLUOP UQPBX
UFLN JWBZLUZBK BJJKUZBLUNFX UF LSP BWPB NG YKNZCZSBUFX EULS
LSP YKNZCZSBUF ZSBKKPFMP. BZZNHJBFUPQ YT PRJPWLX BFQ
ZNBZSPX, LSP XLIQPFLX MN LSWNIMS LSP HNXL UHJNWLBFL XLPJX
UF ZWPBLUFM B XNKILUNF ZNFZPJL, UFZKIQUFM LPZSFUZBK
JWNLNLTJPX. LSP ZNIWXP UX NGGPWPQ BX B LEN EPPC YKNZC
XPHUFBW, LBCUFM JKBZP YPGNWP LSP XLBWL NG LSP GBKK XPHPXLPW.
```

Decrypt it using frequency analysis and by exploiting the relations between
letters.

Hint: Use frequency analysis only to decipher the three most frequent letters.
There are online tools that can help you find the relative frequencies of the
letters, for example <https://www.101computing.net/frequency-analysis/>.
